# 试卷信息收集

## 目标
收集初中 7、8、9 年级各地区期中、期末考试试题及答案

## 目录结构

```
exams/
├── grade7/
│   ├── mid-term/     # 七年级期中
│   │   ├── beijing/
│   │   ├── shanghai/
│   │   ├── guangdong/
│   │   ├── jiangsu/
│   │   └── ...
│   └── final-term/  # 七年级期末
├── grade8/
│   ├── mid-term/     # 八年级期中
│   └── final-term/  # 八年级期末
└── grade9/
    ├── mid-term/     # 九年级期中
    └── final-term/  # 九年级期末
```

## 收集要求

### 1. 内容格式
每个试卷文件包含：
- 试卷标题（年份+地区+学期+年级+科目）
- 考试时间
- 满分分值
- 试题部分
- 答案部分

### 2. 命名规范
```
{年份}_{地区}_{学期}_{年级}_{科目}.md

示例：
2025_北京_期中_七年级_语文.md
2024_上海_期末_八年级_数学.md
```

### 3. 来源推荐
- 学科网 (zxxk.com)
- 中考网 (zhongkao.com)
- 高考资源网 (ksmb.com)
- 各省市教育局官网
- 学校官网

### 4. 重点科目
- 语文、数学、英语（必选）
- 物理、化学（八九年级）
- 历史、道法（各地不同）

### 5. 优先级
1. 🔴 高：北上广深一线城市试卷
2. 🟡 中：省会城市试卷
3. 🟢 低：其他城市试卷

## 整理流程
1. 网上搜索下载试卷 → 2. 整理成统一格式 → 3. 放到对应目录 → 4. 定期导入 data/
