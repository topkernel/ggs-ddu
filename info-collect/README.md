# 信息收集目录

## 工作流程

```
1. 网上搜集信息
       ↓
2. 按分类整理到 info-collect/目录
       ↓
3. 审核、编辑、校对
       ↓
4. 合并到 data/ 具体数据文件
```

## 目录结构

```
info-collect/
├── README.md                    # 本文件
├── grade7b/                     # 7年级知识点
│   ├── chinese/
│   ├── math/
│   ├── english/
│   ├── history/
│   └── geography/
├── grade8b/                     # 8年级知识点
│   ├── math/
│   └── physics/
├── grade9/                      # 9年级知识点
│   └── chemistry/

    ├── README.md                # 试卷收集说明
    ├── grade7/
    │   ├── mid-term/            # 期中
    │   └── final-term/         # 期末
    ├── grade8/
    │   ├── mid-term/
    │   └── final-term/
    └── grade9/
        ├── mid-term/
        └── final-term/
```

## 命名规范

- 知识点文件：`知识点名称.md` 或 `单元名称.md`
- 试卷文件：`{年份}_{地区}_{学期}_{年级}_{科目}.md`
  - 例：`2025_北京_期中_七年级_语文.md`

## 收集来源

- 教材内容
- 教辅资料
- 网上公开教育资源
- 老师备课笔记
- 学科网 (zxxk.com)
- 中考网 (zhongkao.com)

## 注意事项

1. 收集的信息需要注明来源
2. 重要知识点标记"必考"、"重点"
3. 例题需要附带答案解析
4. 试卷必须包含**答案**
5. 定期整理和清理未使用的信息
