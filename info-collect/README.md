# 信息收集目录

## 工作流程

```
1. 网上搜集信息
       ↓
2. 按分类整理到 info-collect/目录
       ↓
3. 审核、编辑、校对
       ↓
4. 合并到 data/ 具体数据文件
```

## 目录结构

```
info-collect/
├── README.md                    # 本文件
├── grade7b/
│   ├── chinese/                 # 7年级下语文
│   │   ├── 古诗文/
│   │   ├── 现代文/
│   │   └── 写作/
│   ├── math/                    # 7年级下数学
│   ├── english/                 # 7年级下英语
│   ├── history/                 # 7年级下历史
│   └── geography/              # 7年级下地理
├── grade8b/
│   ├── math/
│   └── physics/
└── grade9/
    └── chemistry/
```

## 命名规范

- 文件名：`知识点名称.md` 或 `单元名称.md`
- 内容格式：见 `../data/数据格式说明.md`

## 收集来源

- 教材内容
- 教辅资料
- 网上公开教育资源
- 老师备课笔记

## 注意事项

1. 收集的信息需要注明来源
2. 重要知识点标记"必考"、"重点"
3. 例题需要附带答案解析
4. 定期整理和清理未使用的信息
