# 📚 知识点学习平台 - 使用说明

## 🎨 界面优化

### ✅ 已完成的优化

1. **紧凑的顶部工具栏**（仅40px高）
   - 📚 知识点标题
   - 实时显示：年级 | 科目 | 页码
   - ⚙️ 设置按钮

2. **底部导航栏**（仅40px高）- 一行搞定所有功能
   - ⏮ 跳到第一页
   - ⏪ 后退10页
   - ◀ 上一页
   - 📄 页码显示和跳转
   - ▶ 下一页
   - ⏩ 前进10页
   - ⏭ 跳到最后一页
   - 📝 导出Markdown
   - 🖨️ 打印/保存PDF

3. **知识点内容区**
   - 真正全屏展示
   - 无内部滚动条
   - 使用浏览器原生滚动
   - 内容自然撑开页面

4. **手势操作**
   - ✅ 左右滑动翻页
   - ✅ 键盘方向键导航
   - ✅ Home/End跳转首尾
   - ✅ ESC关闭设置面板

## 🎯 快捷键

| 按键 | 功能 |
|------|------|
| `←` | 上一页 |
| `→` | 下一页 |
| `Home` | 跳到第一页 |
| `End` | 跳到最后一页 |
| `ESC` | 关闭设置面板 |

## 📱 手势操作

- **向左滑动**：下一页
- **向右滑动**：上一页

## 💾 导出功能

### 📝 导出Markdown
点击底部工具栏的 `📝` 按钮，将当前知识点导出为Markdown文件。

### 🖨️ 打印/保存PDF
点击底部工具栏的 `🖨` 按钮，可以：
- 打印当前知识点
- 在打印对话框中选择"保存为PDF"

## ⚙️ 设置面板

点击右上角的"⚙️ 设置"按钮，可以：
1. **选择年级**：2年级、7年级、8年级
2. **选择科目**：
   - 📖 语文 (90个知识点)
   - 🔢 数学 (50个知识点)
   - 🇬🇧 英语 (24个知识点)
   - 📜 历史 (25个知识点)
   - 🌍 地理 (20个知识点)
