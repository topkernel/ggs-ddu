# 7年级下学期数学知识点库

## 📚 概述

本项目为7年级下学期数学科目系统补充知识点，按照教材章节整理，包含完整的知识点结构。

## 📁 文件说明

### 1. `knowledge-base-grade7b-math-final.js`
**主要知识点文件** - 包含核心知识点示例（约50个详细知识点）

**特点：**
- ✅ 格式完整，可直接使用
- ✅ 每个知识点包含：概念定义、公式定理、例题讲解、易错点
- ✅ 涵盖所有六个章节的核心内容
- ⚠️ 需要补充完整（当前约50个，目标270个）

### 2. `knowledge-base-grade7b-math-complete.js`
**完整知识点文件** - 包含第一章完整45个知识点

**特点：**
- ✅ 第一章（相交线与平行线）完整，45个知识点
- ✅ 每个知识点详细完整
- ⚠️ 其他章节需要补充

### 3. `generate-math-knowledge.js`
**知识点生成器** - 可扩展的生成框架

**用途：**
- 📝 用于批量生成知识点
- 📝 提供标准化的知识点结构
- 📝 便于后续维护和扩展

## 📖 知识点结构

### 每个知识点包含：

```javascript
{
    title: "知识点标题",
    tag: "标签（基础/重点/必考/拓展）",
    tagClass: "样式类（basic/important/advanced）",
    content: `
        <h4>概念定义</h4>
        <ul>
            <li><strong>概念名：</strong>定义说明</li>
        </ul>
        <div class="advanced">
            <h4>⭐ 重要定理/公式</h4>
            <div class="formula">
                公式内容
            </div>
            <h4>⚠️ 易错点</h4>
            <ul>
                <li>❌ 常见错误</li>
                <li>✅ 正确做法</li>
            </ul>
            <div class="example">
                <strong>例题：</strong>题目内容<br>
                <button class="show-answer" onclick="toggleAnswer(this)">查看答案</button>
                <div class="answer">
                    答案和解析
                </div>
            </div>
        </div>
    `
}
```

## 🎯 使用方法

### 1. 直接使用
```javascript
// 引入知识点库
const knowledge = require('./knowledge-base-grade7b-math-final.js');

// 访问知识点
console.log(knowledge.math[0]); // 第一个知识点
```

### 2. 与现有文件合并
```javascript
// 读取现有文件
const existingKnowledge = require('./knowledge-base-grade7b.js');

// 合并数学知识点
existingKnowledge.math = [
    ...existingKnowledge.math,
    ...knowledge.math
];
```

### 3. 扩展知识点
使用 `generate-math-knowledge.js` 作为模板，添加更多知识点。

## 📝 补充指南

### 如何补充完整知识点库：

#### 步骤1：确定章节和类型
```
例如：第二章 - 实数 - 平方根
```

#### 步骤2：列出该类型的所有知识点
```
46. 平方根的概念
47. 算术平方根的性质
48. 平方根的计算
49. 平方根的应用
...
```

#### 步骤3：按照模板格式编写
```javascript
{
    title: "知识点编号. 知识点名称",
    tag: "标签",
    tagClass: "样式类",
    content: `
        // 按照模板填写内容
    `
}
```

#### 步骤4：测试和验证
- 确保格式正确
- 确保内容准确
- 确保例题有答案
