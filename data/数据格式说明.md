# 知识点数据格式说明

本文档说明知识点数据的标准格式，以及如何将新的知识点集成到静态页面中。

## 📋 数据格式标准

### 方式一：直接HTML格式（推荐）

这是最简单直接的格式，适合人工编写和维护。

**文件命名**：`grade{年级}{科目}.js`

例如：`grade7b-chinese.js`（7年级下学期语文）

**数据结构**：

```javascript
const grade7bChinese = {
    // 第一单元
    unit1: [
        {
            title: "1. 知识点标题",
            tag: "重点",  // 可选值：必考、重点、基础
            tagClass: "important",  // 对应样式：important（重点）、basic（基础）
            content: `
                <h4>知识点标题</h4>
                <ul>
                    <li><strong>要点1：</strong>详细说明</li>
                    <li><strong>要点2：</strong>详细说明</li>
                </ul>

                <div class="advanced">
                    <h4>⭐ 进阶内容</h4>
                    <p>详细讲解...</p>
                </div>

                <div class="example">
                    <strong>例题：</strong>题目内容<br>
                    <button class="show-answer" onclick="toggleAnswer(this)">查看答案</button>
                    <div class="answer">
                        答案解析内容...
                    </div>
                </div>
            `
        },
        // 更多知识点...
    ],

    // 第二单元
    unit2: [
        // 知识点...
    ],

    // 第三单元
    unit3: [
        // 知识点...
    ]
};
```

## 🔄 工作流程

### 步骤1：准备知识点数据

创建Markdown文件，按照以下格式编写：

```markdown
# 7年级下学期数学知识点

## 第一章：相交线与平行线

### 1.1 对顶角

**标签**：必考

**定义**：
两条直线相交形成的四个角中，有公共顶点且两边互为反向延长线的两个角叫做对顶角。

**性质**：
- 对顶角相等
- 必须有公共顶点
- 两边互为反向延长线

**公式**：
若∠1和∠2是对顶角，则∠1 = ∠2

**例题**：
题目：两条直线相交，其中一个角是35°，求它的对顶角和邻补角的度数。

答案：对顶角是35°（对顶角相等），邻补角是180°-35°=145°

**易错点**：
对顶角必须有公共顶点且两边互为反向延长线，不是所有相对的角都是对顶角。
```

### 步骤2：转换为JavaScript格式

将Markdown转换为标准JavaScript对象：

```javascript
{
    title: "1.1 对顶角的概念与性质",
    tag: "必考",
    tagClass: "important",
    content: `
        <h4>定义</h4>
        <p>两条直线相交形成的四个角中，有公共顶点且两边互为反向延长线的两个角叫做对顶角。</p>

        <h4>性质</h4>
        <ul>
            <li>对顶角相等</li>
            <li>必须有公共顶点</li>
            <li>两边互为反向延长线</li>
        </ul>

        <div class="formula">
            若∠1和∠2是对顶角，则∠1 = ∠2
        </div>

        <div class="example">
            <strong>例题：</strong>两条直线相交，其中一个角是35°，求它的对顶角和邻补角的度数。<br>
            <button class="show-answer" onclick="toggleAnswer(this)">查看答案</button>
            <div class="answer">
                对顶角是35°（对顶角相等），邻补角是180°-35°=145°
            </div>
        </div>

        <div class="advanced">
            <h4>⚠️ 易错点</h4>
            <p>对顶角必须有公共顶点且两边互为反向延长线，不是所有相对的角都是对顶角。</p>
        </div>
    `
}
```

### 步骤3：创建知识点文件

1. **创建文件**：命名为 `grade{年级}{科目}.js`
   - 例如：`grade7b-math.js`（7年级下学期数学）
   - 例如：`grade8a-physics.js`（8年级上学期物理）

2. **定义变量**：使用驼峰命名法
   ```javascript
   const grade7bMath = {
       unit1: [ /* 知识点数组 */ ],
       unit2: [ /* 知识点数组 */ ],
       // ...
   };
   ```

3. **按单元组织**：每个单元一个数组
   - 数组中的每个元素都是一个知识点对象
   - 知识点对象必须包含：title, tag, tagClass, content

### 步骤4：集成到页面

在 `index.html` 中添加引用：

1. **添加script标签**：
```html
<script src="grade7b-math.js"></script>
```

2. **在gradeData中添加数据**：
```javascript
grade7b: {
    chinese: flattenKnowledge(typeof grade7bChinese !== 'undefined' ? grade7bChinese : []),
    math: flattenKnowledge(typeof grade7bMath !== 'undefined' ? grade7bMath : {}).map(kp => convertKnowledgePoint(kp, 'math')),
    // ...
}
```

3. **更新设置面板**（如果需要）：
```html
<option value="math">🔢 数学</option>
```

## 📊 文件组织规范

**知识点文件**：

- 年级：1-9
- 学期：a（上学期）、b（下学期）

## 📞 常见问题

**Q: 如何添加新的年级？**
A: 创建新的知识点文件，在index.html的gradeData中添加对应年级配置，并在设置面板中添加选项。

**Q: 如何处理大量的知识点？**
A: 建议按单元分成多个数组（unit1, unit2等），便于维护和管理。

**Q: 可以混合使用两种数据格式吗？**
A: 可以，但结构化格式需要在index.html中添加转换函数。推荐统一使用直接HTML格式。

**Q: 如何批量生成知识点？**
A: 先用AI工具生成Markdown格式，然后按照步骤2手动转换为JavaScript格式。
