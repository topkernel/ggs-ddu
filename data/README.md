## 🎯 使用方法

### 本地使用
1. 打开 `index.html` 文件
2. 选择年级和科目
3. 使用左右箭头或按钮翻页
4. 点击"查看答案"查看详细解答

### 快捷键
- `←` / `→`：上一页/下一页
- `Home`：跳转到第一页
- `End`：跳转到最后一页

## 💡 学习建议

1. **每天学习5-10个知识点**
   - 语文：2个
   - 数学：2个
   - 英语：1个
   - 历史/地理：每周各2个

2. **复习策略**
   - 第一遍：通读所有知识点
   - 第二遍：重点标记不会的
   - 第三遍：反复练习易错题

3. **时间安排**
   - 完整学习一遍：约1个月
   - 系统复习：约2周
   - 考前冲刺：约1周
